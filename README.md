## About

MSc Computing (Data Analytics) student at Dublin City University with production experience building backend systems, data pipelines, and applied ML. I have shipped a multi-tenant SaaS backend serving 1,000+ daily requests, a live analytics platform for Irish public data, and a multi-agent AI system on Google Cloud Run. First author of a published paper on real-time ML for accessibility.

My current MSc practicum, SmartDisagg, is the first documented NILM system built for Ireland's ESB smart meter format. It disaggregates 30-minute meter data into appliance-level usage across 1,961 real households, validated against a commercial energy-insights product and independent EV telemetry.
